In order to use the project, it is necessary to have installed the ObjectBox and JavaFX libraries.

The simplest is to use IntelliJ and to import the project via GitHub. The build.gradle file will automatically import
the necessary parameters to use the code.

To follow the manipulations of the report, modulate the values written in the Main.java file .
The javaFX interface is only there to give an idea of how to use ObjectBox as explained in the report. If a will is to
use the application certe incomplete, restart the code in the hand by uncommenting the lines 33 and 34 as well as the
line 60 of the Generator.java file in order to generate 15 Items per person.
